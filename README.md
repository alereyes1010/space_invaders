# Space-Invaders-Project
Proyecto universitario desarrollado en C++ utilizando la librería SFML, como parte del curso de Programación I. Inspirado en el clásico juego Space Invaders, implementa mecánicas básicas de disparo, movimiento y colisiones. Incluye gestión de sprites, detección de colisiones y utiliza programación orientada a objetos y renderizado con SFML.
